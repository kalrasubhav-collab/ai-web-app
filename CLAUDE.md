# AI Web App

## Project Goal
A simple web app where users can type text and get an AI-powered response.
Built with Flask (Python) and the Anthropic API.

## Tech Stack
- Python 3
- Flask (web framework)
- Anthropic Python SDK (for Claude AI responses)
- Plain HTML/CSS (no frameworks)

## Features to Build
1. A clean home page with a text input form
2. User submits text and gets an AI response back
3. Response displays nicely on the page
4. Simple, clean design

## Code Style
- Simple, readable Python
- Functions for each route
- Clear variable names
- Comments explaining what each section does

## Project Structure
- app.py — main Flask application
- templates/ — HTML files
- static/ — CSS files

## Commands
- Run app: python app.py
- App runs on: http://localhost:5000

## What NOT to do
- No complex JavaScript frameworks
- No databases yet
- Keep it simple and clean
